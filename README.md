<br>Implementation Guide 
<br>1 - Project
<br>1 - Open the Project in your android studio;
<br>2 - *IMPORTANT* Change the Package Name. (https://stackoverflow.com/questions/16804093/android-studio-rename-package)

<br>2 - Firebase Panel
<br>1 - Create Firebase Project (https://console.firebase.google.com/);
<br>2 - Import the file google-service.json into your project
<br>3 - Connect to firebase console authentication and database from your IDE (video 2)
<br>4 - For sending notification, paste your Firebase project key into your project APIService.java (video 18) 
<br>5 - When you change database settings, you likely will need to uninstall and reinstall apps to avoid app crashes due to app caches.
